## Overview

Number Guessing Game is a simple console-based Java application where the player tries to guess a randomly generated number within a limited number of attempts.

The game provides hints to guide the player until the correct number is guessed.

---

## Problem Statement

Beginner programmers often need practical ways to understand:

- Random number generation
- Conditional logic
- User input handling
- Loop control

A basic interactive game helps strengthen these core programming concepts.

---

## Solution

This game generates a random number and allows the user to guess it.

The system provides feedback such as:

- Too high
- Too low
- Correct guess

This makes the learning process interactive and engaging.
